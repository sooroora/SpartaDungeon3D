## 💜 주요기능
+ Player InputManager
  - 이동, 벽 등반

+ Camera CamerController
  - 장애물 탐지, 1인칭 3인칭 전환

+ Inventory System
  - ScriptableObject 를 통해 Item 종류 관리
  - Inventory 및 Item class 를 통한 실제 Item Data 관리
  - 사용 및 장비

+ IInteractable
  - 상호작용 가능한 일반 오브젝트 및 떨어진 아이템을 통합하여 관리

+ NavMesh 를 이용한 자동 이동
